# Krebs Cycle Simulation

A Python simulation of the Krebs cycle (citric acid cycle) built in 
Jupyter Notebook, designed to model enzymatic reactions and metabolic 
regulation under different conditions.

## What it simulates
- Enzymatic reactions as Python functions
- Metabolite pools as dynamic lists
- Cofactor cycling (NAD+/NADH, CoA)
- CO2 release at each decarboxylation step

## Goals
- Simulate substrate availability and its effect on cycle flux
- Model regulatory conditions (cofactor depletion, substrate limitation)
- Build towards a complete 8-enzyme cycle simulation
